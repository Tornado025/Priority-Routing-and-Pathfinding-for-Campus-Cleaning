# DSA Lab EL Project

To run this project, please follow the below commands:

1. Compile the C program into a shared library:

    ```bash
    make all
    ```

2. Install dependencies for the Python GUI:
    
    ```bash
    cd gui && pip install -r requirements.txt
    ```

3. Run the GUI:

    ```bash
    cd gui && python run gui.py
    ```
